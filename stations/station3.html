Color Wheel Relationships

Prof. Teeters drew a simple color wheel and my mind exploded. There are actual RULES for picking colors that work together! It's not guessing or "good taste" - it's geometry!

The color wheel in HSL terms:

Remember from Station 1: Hue is 0-360¬∞ around the color wheel. 0¬∞ = red, 120¬∞ = green, 240¬∞ = blue, back to 360¬∞ = red again. Once you know your primary hue, you can calculate harmonious colors mathematically!

Complementary harmony (opposites attract):

Colors 180¬∞ apart on the wheel. They create maximum contrast and energy!

Complementary colors formula:
:root {
  --primary: hsl(210, 60%, 50%);         /* Blue (210¬∞) */
  --complement: hsl(30, 60%, 50%);       /* Orange (210¬∞ + 180¬∞ = 390¬∞, wraps to 30¬∞) */
}

The trick: 210 + 180 = 390, but the wheel only goes to 360, so 390 - 360 = 30. Boom! Blue and orange!

Prof. Teeters' tip:

Complementary colors are INTENSE together. Don't use them 50/50! Use one as primary (60-70% of the design) and the other as accent (10-20%). Otherwise it's too much energy!

Analogous harmony (neighbors):

Colors next to each other on the wheel (usually ¬±30¬∞ to ¬±60¬∞). They create calm, cohesive palettes.

Analogous colors formula:
:root {
  --primary: hsl(210, 60%, 50%);         /* Blue (210¬∞) */
  --analogous-1: hsl(180, 55%, 45%);     /* Blue-green (210¬∞ - 30¬∞ = 180¬∞) */
  --analogous-2: hsl(240, 55%, 55%);     /* Blue-purple (210¬∞ + 30¬∞ = 240¬∞) */
}

Analogous palettes feel harmonious because the colors share a "color family." All blues, all greens, all warm tones. Prof. Teeters said "analogous is the safest harmony for beginners - it's almost impossible to mess up!"

Triadic harmony = three colors evenly spaced at 120¬∞ apart (red/yellow/blue, or 0¬∞/120¬∞/240¬∞).

Split-complementary = your primary plus the two colors on either side of the complement (¬±30¬∞ from the complement).

üü† Harmonious palette using analogous colors (much calmer!)

‚úÖ RIGHT: Reduced saturation + neutrals

:root {
  /* Analogous harmony: blue family, reduced saturation */
  --primary: hsl(210, 60%, 50%);         /* Blue - main brand color */
  --secondary: hsl(190, 50%, 45%);       /* Blue-green - supporting color */
  --accent: hsl(230, 55%, 55%);          /* Blue-purple - accent color */

  /* Neutrals for balance (not counted as "colors" in the palette) */
  --text: hsl(0, 0%, 20%);               /* Dark gray text */
  --background: hsl(0, 0%, 98%);         /* Near-white background */
  --border: hsl(0, 0%, 85%);             /* Light gray borders */
}

.header { background: var(--primary); }
.sidebar { background: var(--secondary); }
.button { background: var(--accent); }
.text { color: var(--text); }
body { background: var(--background); }
/* Now it's cohesive, not chaotic! */

See the difference? The hues are related (190-210-230, all in the blue family). The saturations are moderate (50-60%, not 100%). And we added neutrals to let the eyes rest!

Use 60-30-10 for distribution: primary ~60%, secondary ~30%, accent ~10% (neutrals don't count).

Color Temperature and Psychology

Colors aren't just pretty - they communicate meaning! Prof. Teeters showed me how banks use blue (trust), restaurants use red/orange (appetite), and eco-brands use green (nature). My random color choices were sending mixed messages!

Warm vs cool colors:

Warm (0-60¬∞): Energetic, urgent, exciting. Advance toward the viewer. Increase heart rate and appetite. Great for CTAs and action elements.

 

Cool (180-300¬∞): Calm, trustworthy, stable. Recede from the viewer. Lower heart rate and create calm. Great for large areas and backgrounds.

Prof. Teeters' trick:

Warm colors for CTAs (call-to-action buttons) because they demand attention. Cool colors for backgrounds and large areas because they create calm space!

Common color meanings (Western):

RED (0¬∞): Urgency, passion, danger, energy, appetite (use for: errors, sales, food, warnings)

ORANGE (30¬∞): Enthusiasm, creativity, fun, affordability (use for: CTAs, youth brands, calls to action)

YELLOW (60¬∞): Optimism, caution, happiness, attention (use for: warnings, highlights, cheerful brands)

GREEN (120¬∞): Nature, growth, money, health, safety (use for: eco-brands, finance, health, success messages)

BLUE (210¬∞): Trust, professionalism, calm, stability (use for: corporate, tech, healthcare, backgrounds)

PURPLE (270¬∞): Luxury, creativity, spirituality, sophistication (use for: premium brands, beauty, creative fields)

üåç Cultural Warning

These are WESTERN associations! Red means luck in Chinese culture. White means purity in Western culture but mourning in some Eastern cultures. Always research your audience!

The biggest lesson:

Don't pick colors because YOU like them. Pick colors because they communicate the right message to YOUR USERS!

Using temperature contrast for visual hierarchy:
:root {
  /* Cool dominant color (60-70% of design) - creates calm space */
  --primary: hsl(210, 40%, 50%);         /* Cool blue - trust, calm */
  --primary-light: hsl(210, 40%, 95%);   /* Very light blue background */

  /* Warm accent color (10-20% of design) - draws attention */
  --accent: hsl(25, 75%, 55%);           /* Warm orange - energy, action */
  --accent-hover: hsl(25, 75%, 45%);     /* Darker orange for hover */

  /* Neutrals */
  --text: hsl(0, 0%, 15%);
  --background: hsl(0, 0%, 100%);
}

/* Cool colors for large areas (calm, recede) */
.header { background: var(--primary); }
.content-section { background: var(--primary-light); }

/* Warm colors for actions (energy, advance) */
.cta-button { background: var(--accent); }
.cta-button:hover { background: var(--accent-hover); }
.important-link { color: var(--accent); }

See how the cool blue creates a calm foundation, and the warm orange accents pop forward? Your eye naturally goes to the orange CTAs! That's intentional temperature contrast!

‚ôø Accessibility Reminder

When swapping between warm and cool accent colors, always recheck contrast ratios! Developers often brighten oranges or yellows for energy and accidentally drop below WCAG AA standards (4.5:1 for normal text). Use browser DevTools or a contrast checker to verify.
